# Space RTS

> 基于 Godot 4 的太空即时战略游戏（2D 俯视视角）
>
> 类《家园》(Homeworld) 风格，支持框选单位、多武器系统、建筑生产、经济体系、多阵营对战、小地图导航等完整 RTS 玩法。

## ✨ 功能特性

### 🚁 核心 RTS 操控

| 操作 | 说明 |
|------|------|
| 框选 | 左键拖拽矩形框选己方单位 |
| 单击选中 | 左键单击选中单个单位 |
| 双击选中同类 | 双击单位选中全场同类单位 |
| Shift+点击反选 | Shift+左键点击已选中友军取消选中 |
| 右键移动 | 右键空地 → 移动 |
| 右键攻击 | 右键点击敌方 → 锁定攻击 |
| A + 左键攻击移动 | 移动到目标点，沿途自动攻击敌人 |
| A + 左键区域攻击 | 对指定半径范围内敌人自动攻击 |
| W + 左键环绕 | 右键环绕目标飞行（W+拖拽设定半径） |
| Shift+右键指令队列 | 连续下达移动/攻击指令 |
| 反打机制 | 被攻击时自动反击攻击者 |
| 编队 | Ctrl+数字编组，按数字键选中，双击跳转镜头 |
| Ctrl+A 全选 | 选中所有己方战斗单位 |
| T 同类选中 | 选中与当前相同类型的单位/建筑 |
| H 镜头归中 | 镜头移至当前选中单位 |
| F 镜头跟随 | 镜头跟随当前选中单位 |

### ⚔️ 战斗系统

| 特性 | 说明 |
|------|------|
| 4 种武器 | 子弹（BULLET）、导弹（MISSILE）、激光（LASER）、近防炮（PD） |

### 设计要点

- **数值集中**：`game_config.gd` 统一管理所有游戏数值，方便平衡调整
- **Weapon 为 Resource**：纯数据无节点开销，支持工厂方法创建
- **阵营动态生成**：根据配置动态生成阵营，正多边形布局，随机中文名
- **静态统计**：使用 `static var` 跨实例共享阵营积分和 DPS 数据
- **脏标记渲染**：单位仅在状态变化时重绘，提升性能
- **RefCounted 子系统**：输入、选择、阵型、生成等逻辑拆分为独立 `RefCounted` 子脚本
- **状态机设计**：采矿船使用状态机（IDLE→MOVING→MINING→RETURNING→DEPOSITING）
- **AI 策略化**：指挥官偏好 + 多种战术（偷袭/防守/骚扰）+ 经济生产自动调度
- **渐变着色器**：弹体使用自定义 Shader 实现中心亮白→边缘黄色的辉光效果
- **阵营染色**：使用 Shader 仅对飞船中心区域进行阵营色染色，保留原始纹理细节

## 🚀 使用说明

1. 用 **Godot 4** 打开 `project.godot`
2. 运行 `main.tscn`
3. 游戏将自动全屏启动

### 配置阵营

编辑 `game_config.gd` 底部的 `faction_config`：

```gdscript
static var player = [0, 2, 2, 2, 1]     # 玩家阵营
static var f0 = [10, 0, 0, 1, 0]         # AI 阵营模板
static var faction_config: Array = [player, f0, f0, f0]
# [随机船, 护卫舰, 驱逐舰, 巡洋舰, 战列舰]
```

### 配置经济

```gdscript
const INITIAL_MINERALS := 9000.0         # 初始矿物
const SHIPYARD_COST_DRONE := 50          # 各船型造价
const SHIPYARD_COST_BATTLESHIP := 800
const DEPLOY_COST_SHIPYARD := 500        # 部署船厂消耗
const DEPLOY_COST_MINE := 300            # 部署矿场消耗
```
