# Space RTS

> 基于 Godot 4 的太空即时战略游戏（2D 俯视视角）
>
> 类《家园》(Homeworld) 风格，支持框选单位、多武器系统、多阵营对战、小地图导航等核心 RTS 操作。

---

## ✨ 功能特性

### 核心 RTS 操控

| 操作 | 说明 |
|------|------|
| 框选 | 左键拖拽矩形框选己方单位 |
| 点击选 | 左键单击选中单个单位 |
| 右键移动 | 右键空地 → 移动 |
| 右键攻击 | 右键点击敌方 → 锁定攻击 |
| A + 左键攻击移动 | 移动到目标点，沿途自动攻击敌人 |
| A + 左键区域攻击 | 对指定半径范围内敌人自动攻击 |
| 反打机制 | 被攻击时自动反击攻击者 |
| 编队 | Ctrl + 1~9 编组，按数字键选中 |

### 战斗系统

| 特性 | 说明 |
|------|------|
| 4 种武器 | 子弹（BULLET）、导弹（MISSILE）、激光（LASER）、近防炮（PD） |
| 武器槽位 | 最多 8 个槽位，独立冷却、独立旋转、独立瞄准 |
| 子弹物理 | 带质量 / 加速度 / 最高速度的物理运动 |
| 追踪导弹 | 自动追踪目标，持续转向，可被 PD 拦截 |
| PD 近防 | 自动拦截敌方追踪弹体 |
| 激光 | 连续光束攻击（3 层辉光效果），按脉冲周期持续伤害 |
| 护盾 / 结构 | 双血条系统，护盾自动恢复，结构受击永久损失 |
| 5 种船型 | 无人机 → 护卫舰 → 驱逐舰 → 巡洋舰 → 战列舰，尺寸/属性递增 |

### 阵营系统

| 特性 | 说明 |
|------|------|
| 多阵营 | 通过 `game_config.gd` 配置阵营数量和舰队组成 |
| 随机阵营名 | 每次开局自动生成中文阵营名称 |
| AI 指挥官 | 每个 AI 阵营有独立指挥官偏好（进攻/防御/平衡） |
| 积分榜 | 击杀敌方单位获得阵营积分 |
| 武器 DPS 统计 | 实时统计各阵营各武器的伤害输出，暂停菜单查看 |

### 技能系统（数字键 1~6）

| 技能 | 键位 | 效果 |
|------|------|------|
| 加速 | 1 | 2 倍移速，持续 10s |
| 速射 | 2 | 5 倍射速，持续 10s |
| 减伤 | 3 | 减免 90% 伤害，持续 10s |
| 跃迁 | 4 | 瞬间跳跃至多 3000px |
| 减速光环 | 5 | 自动释放，对周围 1000px 内敌方施加减速 |
| 净化 | 6 | 清除自身 debuff 并获得 5s 免疫 |

### 相机与地图

| 特性 | 说明 |
|------|------|
| 滚轮缩放 | Camera2D 平滑缩放 |
| 边缘滚动 | 鼠标移到屏幕边缘自动平移 |
| 小地图 | 右下角动态小地图，点击 / 拖拽跳转位置 |
| 无限地图 | 无边界限制，小地图动态适配所有单位 |

### UI 系统

| 特性 | 说明 |
|------|------|
| 顶部 HUD | 飞船数 / 弹体数 / FPS 实时显示 |
| 暂停菜单 | ESC 打开——阵营数据（积分 + DPS）、继续 / 重新开始 / 退出 |
| 游戏结束 | 胜负判定，显示获胜阵营 |
| 选中高亮 | 选中单位有圆形指示器 |
| 血条 / 护盾条 | 头顶显示 |

## 🏗️ 技术架构

### 目录结构

```
project.godot
├── assets/                  # 资源文件（SVG 导入配置）
│   ├── ship.svg / ship2.svg
│   ├── cruiser.svg / destroyer.svg / frigate.svg / drone.svg
│   ├── missile.svg
│   └── weapon_launcher/     # 炮塔纹理
├── scenes/
│   ├── main.tscn            # 主场景（游戏入口）
│   ├── unit.tscn            # 单位预制体
│   ├── projectile.tscn      # 弹体预制体
│   ├── overlay.tscn         # 暂停覆盖层 UI
│   └── minimap.tscn
├── scripts/
│   ├── game_config.gd       # 全局数值配置
│   ├── main.gd              # 主循环（输入/相机/单位管理/AI 调度）
│   ├── unit.gd              # 单位核心（属性/战斗/炮塔/技能/绘制）
│   ├── unit_combat.gd       # 战斗逻辑（武器更新/PD 拦截）
│   ├── unit_movement.gd     # 移动逻辑（追逐/环绕/编队）
│   ├── weapon.gd            # 武器数据（Resource 类型）
│   ├── projectile.gd        # 弹体（物理/追踪/伤害）
│   ├── hud.gd               # 顶部 HUD 调试栏
│   ├── minimap.gd           # 小地图
│   ├── overlay.gd           # 暂停/菜单 UI
│   └── ai_controller.gd     # AI 控制器
└── README.md
```

### 设计要点

- **数值集中**：`game_config.gd` 统一管理所有游戏数值
- **Weapon 为 Resource**：纯数据无节点开销
- **阵营动态生成**：根据配置动态生成阵营、正多边形布局
- **静态统计**：使用 `static var` 跨实例共享阵营积分和 DPS 数据
- **脏标记渲染**：单位仅在状态变化时重绘，提升性能

## 🚀 使用说明

1. 用 **Godot 4** 打开 `project.godot`
2. 运行 `main.tscn`
3. 游戏将自动全屏启动

### 配置阵营

编辑 `game_config.gd` 底部的 `faction_config`：

```gdscript
static var faction_config = [
    [10, 0, 0, 0, 0],   # 阵营0（玩家）：10 无人机
    [0, 2, 2, 2, 1],    # 阵营1（AI）
]
# [随机, 护卫舰, 驱逐舰, 巡洋舰, 战列舰]
```
