# Dev Log

start at 2026/06/22

## ✅ Day 01

### 核心 RTS 操控
| 功能 | 描述 |
|------|------|
| 框选 | 左键拖拽矩形框选己方单位 |
| 点击选 | 左键单击选中单个单位 |
| 右键移动 | 右键空地 → 移动 |
| A+左键攻击移动 | 移动到目标点，沿途自动攻击敌人 |
| A+左键区域攻击 | 对指定半径范围（500px）内敌人自动攻击 |
| 右键攻击 | 右键点击敌方 → 锁定攻击 |
| 反打机制 | 被攻击时自动反击攻击者 |
| ESC 暂停菜单 | 暂停/继续/重新开始/退出 |

### 战斗系统
| 功能 | 描述 |
|------|------|
| 4 种武器 | 子弹（BULLET）、导弹（MISSILE）、激光（LASER）、近防炮（PD） |
| 武器槽位 | 最多 8 个槽位，独立冷却、独立旋转 |
| 槽位偏移 | 每个槽位有位置偏移（上/下/左/右/对角线） |
| 槽位独立瞄准 | 每个炮塔从槽位实际位置计算瞄准角度 |
| 子弹物理 | 带质量/加速度/最高速度的物理运动 |
| 追踪导弹 | 自动追踪目标，持续转向 |
| PD 拦截 | 近防炮自动拦截敌方追踪弹 |
| 激光 | 连续光束（3 层辉光效果） |
| 护盾/结构 | 双血条系统（护盾自动恢复，结构受击永久损失） |

### 单位系统
| 功能 | 描述 |
|------|------|
| 队伍系统 | BLUE（玩家）/ RED（AI） |
| 碰撞分离 | 单位之间自动排斥避免重叠 |
| 飞船朝向 | Sprite2D 旋转跟随移动速度方向 |
| 选中高亮 | 选中单位有视觉效果 |
| 血条/护盾条 | 头顶显示，护盾蓝条在上，结构彩色条在下 |

### 相机与地图
| 功能 | 描述 |
|------|------|
| 滚轮缩放 | Camera2D 平滑缩放 |
| 边缘滚动 | 鼠标移到屏幕边缘自动平移 |
| 小地图 | 右下角动态小地图，点击/拖拽跳转 |
| 无限地图 | 无边界限制，小地图动态适配 |
| 全屏模式 | 启动自动全屏 |

### 环绕模式（W）
| 功能 | 描述 |
|------|------|
| W+左键 | 选中单位环绕目标飞行 |
| 环绕方向 | 根据切入位置自动选择顺时针/逆时针 |
| 角速度 | 自动根据半径计算，保证飞船能追上轨道 |
| 取消环绕 | 右键移动或新指令取消 |
| 轨道绘制 | 调试时可看到轨道圆 |
